Design
As a model for my portfolio I used C.R.A.P.

In order to create the C, contrast, I added headlines with different sizes to create a hierarchy. I also made movement contrast such as putting the text in a Z-form, thereby helping the reader through the design (source).

R stands for repetition which I created by using the same pictures on the frontpage, the project page and the flow pages.

A is for alignment. On the front page I tried position the picture of myself using The Golden Ratio.

P stands for proximity and describes grouping elements that relate one way or another. Iâ€™ve done that by placing each picture next to the text it belongs to.

I wanted my portfolio to be simple and easy to navigate. Therefore, I added space between the columns and used pictures to separate blocks of text.

I have decided to keep the same blue color as in my earlier portfolio version. Blue represents trust and dependability, values that I would like to be associated with. Additionally, Iâ€™ve tried to keep the design simple and calm by using light colors and avoid distractions by using only a few number of colors. The simple design is also shown by aligning pictures and text together.
